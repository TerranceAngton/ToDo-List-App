# To-Do List App

**Organize your tasks quickly and efficiently with this interactive web-based To-Do List.**

A simple, interactive To-Do List web app built with HTML, CSS, and JavaScript.

## Features

- Add tasks with a title, due date, and priority (Low, Medium, High)  
- Edit and delete tasks  
- Mark tasks as completed  
- Data saved in browser localStorage

## How to Use

1. Open `index.html` in a web browser.  
2. Type a task, choose a due date and priority, then click **Add**.  
3. Tasks appear in the list below.  
4. Use **Edit** to change a task, **Delete** to remove it, or check the box to mark it as completed.

## Tech Stack

- HTML  
- CSS  
- JavaScript (vanilla)  
- Browser localStorage
